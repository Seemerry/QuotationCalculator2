# LED显示屏智能报价计算器

一个功能完整的LED显示屏报价计算Web应用程序，帮助用户根据需求获取精准的价格估算。

## 功能特点

- 🎯 **智能推荐** - 根据观看距离和使用环境自动推荐最适合的LED型号
- 💰 **多档价位** - 提供经济型、标准型和高端型三种价格选择
- 📱 **响应式设计** - 完美适配手机、平板和桌面设备
- 🏢 **全场景覆盖** - 支持室内外固定安装和租赁使用
- 📊 **实时计算** - 即时显示所需箱体数量和价格范围
- 📧 **报价服务** - 可提交联系信息获取详细报价单

## 支持的使用环境

- **室内固定** - 商场、会议室、酒店等固定室内场所
- **室内租赁** - 展览、活动、演出等临时室内使用
- **户外固定** - 广告牌、建筑外墙等固定户外安装
- **户外租赁** - 演出、赛事、庆典等临时户外活动

## 快速开始

### 在线使用
直接用浏览器打开 `index.html` 文件即可使用。

### 本地运行
```bash
# 使用Python内置服务器
python -m http.server 8000

# 或使用Node.js
npx serve .

# 然后在浏览器中访问
http://localhost:8000
```

## 使用说明

1. **选择使用环境** - 根据实际应用场景选择室内外和固定/租赁
2. **选择偏好** - 根据预算选择经济型、标准型或高端型
3. **选择观看距离** - 选择观众与屏幕的典型观看距离
4. **输入屏幕尺寸** - 输入所需的屏幕宽度和高度（米）
5. **获取报价** - 系统自动推荐最适合的型号并显示价格范围
6. **联系销售** - 如需详细报价，可填写联系信息

## 自定义和扩展

### 添加新产品
在 `script.js` 的 `ledDatabase` 对象中添加新的产品规格：

```javascript
'environment-type': {
    cabinetSize: { width: 0.64, height: 0.48 },
    TW11: [
        { model: '新型号', pixelPitch: 1.2, pricePerSqm: 1000, cabinetSetPrice: 300 }
    ]
}
```

### 修改样式
所有样式都在 `styles.css` 中，采用移动优先的响应式设计。

### 自定义计算逻辑
主要的计算函数在 `script.js` 的 `calculateQuote()` 函数中。
